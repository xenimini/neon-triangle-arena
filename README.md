# NEON TRIANGLE ARENA ⚡

> *«Я не программист. Я просто хотела сделать крутую игру про неоновые треугольники ради колледжа.  
> В итоге я чуть не сошла с ума, 47 раз убивала порт 5174, проклинала Socket.IO и теперь официально ненавижу npm.  
> Но оно работает. Иногда.»*

Маленькая, но очень яркая браузерная мультиплеер-арена на Three.js + Socket.IO.

### Как запустить игру

**Самый простой способ:**

1. Дважды кликни по файлу `start.bat` (если есть)
2. Или вручную:

```bash
# Терминал 1 — сервер
node server.js

# Терминал 2 — клиент (в папке client)
npm run dev

Открой в браузере: http://localhost:5174

Введи имя — и вперёд, жги арену!
Управление

WASD или стрелочки — движение
Клик по экрану — захват курсора (мышь)
Чат работает

Стек

Frontend: Three.js + Vite
Backend: Node.js + Express + Socket.IO
Мультиплеер: реальное время
